Event Booking System
This is a simple event booking system that allows users to book tickets for different events. The system consists of a Flask web application and a PostgreSQL database.

Installation
Clone the repository to your local machine
Install the required dependencies by running pip install -r requirements.txt
Create a PostgreSQL database called event_booking_system
Set the database URI in the config.py file
Run the database migrations by running flask db upgrade
Usage
Start the Flask web application by running flask run
Navigate to the application in your web browser at http://localhost:5000
Users can create an account and login to book tickets for different events
Admin users can create and manage events and venues
Features
User authentication and authorization
Event and venue management for admin users
Ticket booking and management for users
Insight page for admin users to view popularity of venues and shows
Technologies Used
Flask
SQLAlchemy
PostgreSQL
Bootstrap
